Objects are hash tables, they store information in a key-value pair.
They are very similar to dictionaries in Python.
Unlike an array, a Javascript Object does NOT retain order, instead you accessthe value you want by entering its corresponding key.
They can hold a variety of data types, including nested Objects.

Besides key-value pairs for normal data types, you can add in methods!
Methods are essentially functions that are built into a javascript object.
Usually we've called a function and then passed stuff as parameters,
now we will build this function inside of an object, creating a method for that object.
